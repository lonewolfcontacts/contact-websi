Create Your Contact Page

This starter site uses static pages. To enable users to sign up and create pages automatically you'll need a backend (see README). For now you can create a profile by copying profiles/sample-profile.html and profiles/sample-profile.json and renaming them (e.g., profiles/john-doe.html and profiles/john-doe.json).

Manual quick steps

  1. Duplicate profiles/sample-profile.html to profiles/your-name.html
  2. Edit profiles/your-name.json with your details
  3. Upload both files to the same folder in your repo
  4. Open /profiles/your-name.html to see your profile and use the QR code

Future (recommended)

Connect a backend (Firebase / Supabase) so users can sign up and profiles are generated automatically. See README for pointers.
